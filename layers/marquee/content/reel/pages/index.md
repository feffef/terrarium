---
title: The Marquee
description: The Marvel Cinematic Universe, watched in the order the story actually happens — not the order the films came out.
---

Release order and story order in the MCU split apart almost immediately: the
saga's earliest chapter, set in the middle of the Second World War, reached
theatres years after its own in-universe descendants. **The Marquee** re-runs
the whole thing on a different clock — one chapter per film, in the order the
events actually unfold inside the fiction, each paired with an original
illustration rather than a promotional still.

This is a running project, not a finished one. It opens with the first five
chapters of that timeline; more will be added as the run continues.
